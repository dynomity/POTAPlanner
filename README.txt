POTA Planner by VA6DM v1.0.0

Quick start
1. Run POTAPlanner.exe.
2. The bundled Canada park list loads automatically.
3. Use Update from POTA to refresh the Canadian park data, or Update from CSV if needed.
4. Enter a Start and Destination, then select Plan Route.

Features
- Canadian Parks on the Air park catalogue
- Map, search, rarity filters, and Excel export
- Route planning with optional off-route park additions
- Route-plan Excel exports
